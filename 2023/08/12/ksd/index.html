KAGGLE - STABLE DIFFUSION

I am currently studying the STABLE DIFFUSION competition on Kaggle. This is a learning notebook that will be periodically updated.

Competition Task

Create a generative model for image-to-prompt conversion by reverse-engineering the diffusion process for a given image.

Evaluation Metrics

Evaluate using cosine similarity between predicted and actual prompt embeddings. For each image in the test set, you must predict a prompt that translates to a 384-length embedding vector for image generation.

Dataset

Official data for this competition is not provided. We can use the DiffusionDB as a data source.
