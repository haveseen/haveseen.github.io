MACHINE LEARNING PARADIGMS

Machine Learning, according to Tom Mitchell, is defined as follows: “A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.”

Supervised Learning:

Supervised learning is a learning approach where a model is trained using input data paired with corresponding output labels. In supervised learning, a labeled dataset is provided, where each training example consists of two parts: a feature set and a label. The model learns the relationship between inputs and outputs to make predictions. The goal is to enable the model to accurately predict the output for new, unlabeled data.

Unsupervised Learning:

Unsupervised learning is a learning approach in which a model is trained without labeled data. In unsupervised learning, only input data is provided, and the model is tasked with discovering patterns and structures within the data on its own. The objective is to reveal hidden structures within the data through techniques such as clustering, dimensionality reduction, or association rules.

Reinforcement Learning:

Reinforcement learning is a learning approach in which a model learns optimal behavior by interacting with an environment. In reinforcement learning, the model interacts with the environment by taking various actions and adjusting its strategy based on feedback from the environment. The goal is to learn the best action policy by maximizing cumulative rewards.

Reinforcement learning falls between supervised and unsupervised learning. On one hand, it does not have a predefined set of labeled training data, and the algorithm needs to explore the environment to obtain training data. On the other hand, since the environment provides feedback for each action, the training data acquired through exploration can be considered as having labels.

The primary goal of machine learning is to enable computer systems to extract knowledge and patterns from experience so that they can make intelligent decisions and predictions when faced with new data or situations in the future. Each learning paradigm has its unique application domains and methods, and the choice of the appropriate learning approach depends on the specific problem and the available data.
